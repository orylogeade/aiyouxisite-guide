# aiyouxisite-guide

## 项目简介

`aiyouxisite-guide` 是一个用于归档和发布多个独立 HTML 页面的仓库。  
这些页面以静态 HTML 文件的形式存在，彼此之间没有强关联，也不针对任何单一网站或域名。  
本仓库提供了一种结构化的方式，用于存放、索引和浏览这些页面内容。

## 目录说明

```
aiyouxisite-guide/
├── pages/          # 存放所有独立 HTML 页面
├── assets/         # 页面共享的静态资源（如 CSS、图片、字体等）
├── index.html      # 仓库主页，用于导航和页面列表
└── README.md       # 本文件
```

- `pages/`：每个子文件夹或独立文件对应一个 HTML 页面，命名清晰以便快速识别。
- `assets/`：如果多个页面使用相同资源，可统一放置于此，避免重复。
- `index.html`：提供页面索引，方便直接访问仓库 GitHub Pages 时浏览。

## 页面归档说明

每个 HTML 页面均为独立、自包含的静态文件，原则上不依赖外部网络资源（除共享资源外）。  
页面内容可能涉及不同主题或用途，但均以信息展示或功能演示为主。  
归档时请遵循以下规则：

- 文件名应简洁、语义化，避免特殊字符。
- 如有必要，可在页面内添加注释说明其用途或来源。
- 非必要不引入外部脚本或样式，以保持离线可访问性。

## 维护说明

- 本仓库由维护者不定期更新，欢迎提交 Issue 或 Pull Request 提出改进建议。
- 新增页面时，请放置于 `pages/` 目录下，并更新 `index.html` 中的导航链接。
- 如删除或重命名页面，请同步更新索引文件，避免死链接。
- 不建议在此仓库中存放大量动态内容或需要后端支持的文件。
